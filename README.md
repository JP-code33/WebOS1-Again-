# WebOS1-Again-
I have to redo this project because the last one got detached from my account!

This webOS is made for the mission in the Stardance Challenge. It was to make a simple webOS which is personal to you. They had a guide which I followed and it came out pretty good.

The OS contains a working clock which also has a seconds timer, it has the name of the OS in the top-left corner, a side-dock which contains the app icons like browser. To make it draggable and closeable I have added circle style button which can close the tab. If you want to re-open it then just press the icon on the main screen.

It is built with HTML and JavaScript.

If you want to see the webOS in action then just click on the official deployment link which is found in the GitHub of mine or just use this link: https://jp-code33.github.io/WebOS1-Again-/

Thank you for testing my webOS!
